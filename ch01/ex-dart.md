# 1.1 Running a Dart application
 
## Example Code

Use Dart to print a message on screen.

```dart
void main() {
  print('Hello, Dart World!');
}
```

To run a dart application from the command line:

```dart
dart main.dart
```

## Example Output:

The result of the print statement is shown below:

```dart
Hello, Dart World!
```
