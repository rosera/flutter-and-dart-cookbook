# 1.5 Declaring a String variable
 
## Example Code

```dart
void main() {
  String myVariable = "I am a string";

  String myVariable2 = """
I am a multiline
string
""";

  print(myVariable);
  print(myVariable2);
}
```

## Example Output:

The result of the print statement is shown below:

```dart
I am a string
I am a multiline
string
```
